Welcome to gendata’s documentation!

gendata is a package that provides an API for reading and operating on genetic data in python. It is capable of loading genetic data in PLINK’s bed/bim/fam format, performing simple operations on this genetic data, and saving genetic data objects to bed/bim/fam file format.

Check out the Usage section for further information.
